# What's New in release/1.3

## Highlights

This release focuses on giving you more control over how you view your data. The dashboard now supports filtering by date range and status, so you can zero in on exactly what you need. We've also fixed several issues that affected map performance when monitoring large areas, and improved the overall responsiveness of the interface.

## New Features

- **Dashboard filters** — You can now filter the main dashboard by date range and status. Use the new filter bar at the top of the dashboard to narrow down the data you're viewing.

- **Export to CSV** — Dashboard data can now be exported to CSV format. Click the export button in the top-right corner of any data table.

## Fixes

- **Map performance with many markers** — The map no longer slows down when displaying more than 200 active markers. Scrolling and zooming are now smooth regardless of how many items are visible.

- **Notification timing** — Alerts now appear within 2 seconds of the triggering event, down from the previous 5-10 second delay.

- **Session timeout** — Fixed an issue where users were logged out unexpectedly after 15 minutes of inactivity, even when actively using the application.

## Improvements

- **Faster page loads** — Dashboard and map views now load 40% faster due to optimized data fetching.

- **Better mobile layout** — The sidebar navigation now collapses properly on mobile devices.
